This is a basic application to manage QR codes

QR Codes take a string and translate it into black and white squares that form a grid, which can then be scanned by any modern phone. They are commonly used as ways to point to a web site, encoding a URL so it does not have to be typed by users who see it

Tab 1 (this tab): Read me file

Tab 2: Enter in a string (such as a web site address)

Then you can Preview it and Save it if you like it

At any time on any tab, press Quit to exit program
